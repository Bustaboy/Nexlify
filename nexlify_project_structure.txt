## 📁 Directory Purposes:

### Core Directories:
- **src/**: All source code, organized by functionality
- **gui/**: Desktop GUI application with components
- **api/**: REST API and WebSocket server
- **mobile/**: React Native mobile companion app

### Data & Config:
- **config/**: YAML/JSON configuration files
- **data/**: Database, models, and market data
- **logs/**: Organized logging by category

### Development:
- **tests/**: Comprehensive test suite
- **scripts/**: Utility and maintenance scripts
- **docs/**: All documentation

### Deployment:
- **deployment/**: Docker, K8s, and server configs
- **launchers/**: Various ways to start the app
